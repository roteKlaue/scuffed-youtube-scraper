# Scuffed Youtube Downloader

## Setup

 * Installing dependencies
    ```bash
    npm install
    ```
 * Compiling typescript code
   ```bash
   tsc
   ```
 * Create a .env file with the following properties
    ```env
    OUTDIR = path
    ```
 * Done

## Running

 * Starting
    ```bash
    node .
    ```
 * Paste the link
    ![](./images/paste%20link.PNG)
 * Exiting
    * write nothing
    * write exit
